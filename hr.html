  • Ultra Hr System

    This is a system built using ASP.NET technology, comprising a Web API, an MS SQL Server database, and an Angular web application. It was commissioned by a company looking to replace their existing outdated software with a more user-friendly and feature-rich solution. The system implements authorizations with multiple roles and involves various entities. What sets this HR system apart is that the HR Admin can independently define who needs approval for annual leave or time sheets. The system also includes notifications, both via email and in real-time within the application. The entire software is still in development, but if you are interested in learning more, feel free to reach out, and I'll be happy to share more information with you.
